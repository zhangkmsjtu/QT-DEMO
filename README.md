# QT-DEMO
QT-DEMO  
头文件/源文件切换快捷键：**F4**  

这是一份QT实践案例DEMO.
- demo1:最简桌面  
创建一个窗体，qwedget。
- demo2:登录界面  
创建一个登录窗体，设置登录判断逻辑，登录后跳转窗体，调用QDebug()。
- demo3:倒计时界面  
调出帮助文档：**F1**，调用QTime()，QTimer()。
- demo4:猜数字游戏  
使用时间库qtime。
- demo5: 图片轮播  
图片资源加载，调用QLabel、QPixmap等
- demo6: 数据库  
调用sqlite3工具创建数据库。
- dem07: 信息管理软件  
调用qtableview，实现增删改查。
